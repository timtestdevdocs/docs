## Request Parameters

### Body

<ParamField body="url" type="string">
  The URL of the image containing text to translate. If provided, the image will be downloaded from this URL. Must be a valid URL. Either `url` or
  `file_store_key` must be provided.
</ParamField>

<ParamField body="file_store_key" type="string">
  The file store key of a previously uploaded image. Either `url` or `file_store_key` must be provided.
</ParamField>

<ParamField body="target_language" type="string" required>
  The language code of the target language to translate to. All supported language codes can be found
  [here](https://jigsawstack.com/docs/additional-resources/languages).
</ParamField>

<ParamField body="return_type" type="string" default="url">
  Specifies the format of the returned translated image. Supported values:
  - `url`: Returns a URL to the translated image (default)
  - `binary`: Returns the translated image as binary data
  - `base64`: Returns the translated image as a base64-encoded string
</ParamField>

## Response Structure

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

<ResponseField name="translated_text" type="string">
  The translated text content extracted from the image.
</ResponseField>

<ResponseField name="translated_image" type="string | binary | base64">
  The translated image in the format specified by the `return_type` parameter:
  - If `return_type` is "url": A URL to the translated image
  - If `return_type` is "binary": Binary data of the translated image
  - If `return_type` is "base64": Base64-encoded string of the translated image
</ResponseField>
